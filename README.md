# 🏠 House Price Prediction

A beginner Machine Learning project that predicts house prices based on house area and the number of bedrooms using **Linear Regression**.

---

## 📌 Project Overview

This project uses a Linear Regression model from Scikit-learn to estimate house prices from simple housing data. It demonstrates the basic workflow of a machine learning project:

- Loading data
- Training a model
- Making predictions
- Visualizing data

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Project Structure

```
data-science-project/
│── main.py
│── house_data.csv
│── README.md
```

---

## 📊 Dataset

The dataset contains:

- House Area (sq.ft)
- Number of Bedrooms
- House Price

---

## ✨ Features

- Predicts house prices
- Uses Linear Regression
- Reads data from a CSV file
- Displays a graph using Matplotlib
- Beginner-friendly code

---

## ▶️ How to Run

1. Install the required libraries:

```
pip install pandas numpy matplotlib scikit-learn
```

2. Run the program:

```
python main.py
```
